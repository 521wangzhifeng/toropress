linux下安装golang环境攻略

March 12 , 2013 at 15:51 am - Insion - 0 Comments

step 1：

wget https://go.googlecode.com/files/go1.0.3.linux-amd64.tar.gz

step 2：

tar -C /usr/local -xzf go1.0.3.linux-amd64.tar.gz

step 3：

cd /home
mkdir gopath

cd ~
vim .bashrc

在bash文件的最后添加如下代码来设置目录变量：

export GOROOT=/usr/local/go
export GOARCH=amd64
export GOOS=linux
export GOPATH=/home/gopath
export GOBIN=$GOROOT/bin
export PATH=$GOPATH/bin:$GOROOT/bin:$PATH

保存文件后执行：
source .bashrc

执行env | grep '^GO'测试是否已经正确设置GO的环境变量

Okay！

step 4：

我们来做个测试，输入以下hello world

cat >hello.go <<EOF
package main
import "fmt"
func main() {
fmt.Println("hello, world!")
}
EOF

编译：

go build hello.go

执行：

./hello

输出:hello, world!

 

golang的一些附加工具安装：

 apt-get install git-core
