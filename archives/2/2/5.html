golang安装go-sqlite3 for linux

March 12 , 2013 at 15:33 am - Insion - 0 Comments

直接执行go get github.com/mattn/go-sqlite3会有错误，正确的安装顺序应该按下面来：

如果是在 Mac OS X上面用 Homebrewn的话，请用以下的命令安装:

brew install pkgconfig
brew install sqlite3

go get github.com/mattn/go-sqlite3

 

如果你在用Ubuntu,请用以下的命令安装:

sudo apt-get install pkg-config
sudo apt-get install sqlite3  
sudo apt-get install libsqlite3-dev 

go get github.com/mattn/go-sqlite3


如果你在用 Windows,安装之前需要先装好 pkg-config ：

http://ftp.gnome.org/pub/gnome/binaries/win32/dependencies/

Go does not support static linking for external C library; sqlite3 should be built as a shared library. If it runs on Windows, it needs dll.

Finally, this package can be installed with the go get command:

go get github.com/mattn/go-sqlite3
